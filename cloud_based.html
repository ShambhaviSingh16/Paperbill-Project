Frequently Asked Questions

Find answers to common questions about PaperBill Cloud

Is my data secure in the cloud?

Yes, PaperBill uses bank-grade 256-bit SSL encryption for all data transmission and storage. We perform regular security audits and offer two-factor authentication for added security. Your data is stored in secure data centers with 99.9% uptime guarantee.

Can I access PaperBill Cloud offline?

While PaperBill Cloud is primarily an online solution, we offer limited offline capabilities. You can view recent data and perform some tasks offline, which will sync automatically when you reconnect to the internet.

How often is my data backed up?

Your data is backed up automatically every hour, with daily and weekly snapshots stored in multiple geographic locations for disaster recovery.

Can I migrate from desktop version to cloud?

Yes, we provide free migration tools and assistance to help you move your data from PaperBill Desktop to PaperBill Cloud seamlessly.

What devices can I use with PaperBill Cloud?

PaperBill Cloud works on any device with a modern web browser - Windows PCs, Macs, Linux computers, iPhones, iPads, and Android devices. We also offer dedicated mobile apps for iOS and Android.
